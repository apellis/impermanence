# Impermanence

A timer for meditation retreats. Open source and freely available on the iOS App Store.

## iOS

### Status

* App Store release currently available [here](https://apps.apple.com/us/app/impermanence/id6462615517)
* Source version in this repository: `v1.4.0`

## Android

### Status

* Native Android app now available in this repository (source in [`android/`](android))
* Source version in this repository: `v1.0.0`
* Not yet available on Google Play Store

## Release Notes

### v1.4.0 (iOS) / v1.0.0 (Android)

* Active Day UI refresh with clearer hierarchy while keeping full schedule visible.
* Timeline-oriented day editor improvements, including faster reordering and cleaner controls.
* New setting to keep the screen awake while a day is active.
* Accessibility and readability improvements across day and segment cards.
* Native Android release line established (repository/source distribution); Google Play listing not yet live.
